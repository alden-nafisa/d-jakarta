# D'JAKARTA Website

## Description
D'JAKARTA is a vibrant and informative website dedicated to showcasing the rich history, culture, and modern life of Jakarta. From iconic landmarks like Monas and Kota Tua to traditional Betawi cuisine, this website serves as a comprehensive guide for visitors and enthusiasts alike.

## Features
- Explore Jakarta's history and iconic landmarks.
- Dive into galleries featuring Monas, Kota Tua, and Betawi culture.
- Learn about traditional Betawi cuisine.
- Access event details, maps, and reviews.
- FAQs section to answer common questions.
- Owner biodata for direct contact.
